# Code of Conduct / 行为准则

We want this project to be useful and welcoming for engineers debugging difficult performance problems. Be respectful, specific, and constructive.

我们希望这个项目能帮助大家一起定位复杂性能问题。请保持尊重、具体和建设性。

## Expected Behavior

- Use welcoming and inclusive language.
- Focus criticism on code, data, and reproducible behavior.
- Respect that profiling data can contain sensitive infrastructure details.
- Help newcomers create minimal, sanitized reproductions.

## Unacceptable Behavior

- Harassment, insults, or personal attacks.
- Publishing another party's private profiling data or credentials.
- Repeated off-topic promotion or spam.
- Dismissing reports without explaining what evidence is missing.

## Enforcement

Maintainers may edit, hide, or remove comments and may block participants who repeatedly violate this code of conduct.
